## AI Sandboxes

- [sandvault](https://github.com/webcoyote/sandvault): Run AI agents (Claude Code, OpenAI Codex, and Google Gemini) in a sandboxed macOS user account to protect your system from potential AI-generated code risks.

- [clodpod](https://github.com/webcoyote/clodpod): Run AI agents (Claude Code, OpenAI Codex, and Google Gemini) sandboxed inside a macOS virtual machine for security and isolation.
